## ‚ö° CORE WEB VITALS TARGETS

### LCP (Largest Contentful Paint)
- Target: < 2.5s
- Strategy: Optimize hero images, use Next.js Image

### FID (First Input Delay)  
- Target: < 100ms
- Strategy: Minimize JavaScript, code splitting

### CLS (Cumulative Layout Shift)
- Target: < 0.1
- Strategy: Reserve space for images, avoid layout shifts
